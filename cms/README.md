# CMS Kinesis – bootstrap

Proyecto base del panel de administración (Next.js + Tailwind + React Query).

## Arranque rápido

1. `pnpm install` en la raíz del repo.
2. `cd cms`.
3. `pnpm install` dentro de `cms` (instala Next, React, Tailwind).
4. `pnpm dev`.
5. Abre `http://localhost:3000`.

El backend y la autenticación se configurarán en las siguientes iteraciones de T6.

Nota: el login actual es de prueba y solo guarda los datos en el navegador hasta que se conecte la API real con `X-Admin-Secret`.
